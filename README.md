# Bootleg Snake Game

A classic Snake game built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## How to Play

- Use the **arrow keys** on your keyboard to control the snake
- On mobile, use the **on-screen arrow buttons**
- Eat the 🟦 **blue food** to grow and earn points
- Avoid hitting the **walls** or your **own body**
- Your **high score** is saved automatically via localStorage

---

## Features

- 30×30 grid board powered by CSS Grid
- Real-time score tracking
- Persistent high score via localStorage
- Mobile-responsive with on-screen controls
- Lightweight — pure vanilla JS, no libraries

---

## Setup

No installation or build step needed. Just open it in your browser.
